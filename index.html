Info about Stuartchat

Stuartchat is a local IRC server. You are probably here because you connected to it over Wifi.
It is all hosted on a Raspberry Pi. You can connect to it using an IRC client on PC or phone.
Use the instructions below to access the chat server and chat with everyone else on the wifi.
When you click on the Button below you will get to the Admin panel, you cant access it.

Instructions to connect to the IRC chat

1. Click on the Button below to gain access to the server. A new site will open, its the admin
pannel. You cant access it anyways so just ignore it and close the browser.
2. Download an IRC client for your phone or PC. On Windows I would recommend AdiIRC (you can
download the portable version). Else on Android I would recommend you download CoreIRCLite
3. On the App it will ask for information about your server. You can chose whatever you want,
but the IP is 10.3.141.1 (Server adress) and the Port is 6667. No password required.
4. You can connect to your new server. To start chatting you need to join a channel,
you can do the command /join (#channel name), for example: /join #world
5. You are on the server!
